Understanding Colors  
(C)Cyan (M)Magenta (Y)Yellow (K)Black

Take an image into Photoshop, apply the halftone filter, and print out each color from the separations panel. This separates our image into layers for each color. We'll use these printouts as references for our hand-printed image.


Key Terms

Halftone Screen Angles - Angles which each color is displayed at.

Pointillism Theory - An approach to art which utilizes points to compose an entire image.

Print Color Separation - The separation of color channels visually represented on several printouts.

Photoshop Halftone Effect - This effect is a filter which breaks down an image into separate color channels (CMYK).

Adobe Reflow - A program used to create webpages with a GUI that utilizes creative approach opposed to code created content. 

Responsive Webpages - A webpage that adjusts to different devices' pixel dimensions.
